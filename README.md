# 🚀 Smart Business Analytics Engine

An AI-powered business analytics engine that automatically analyzes any business dataset — EDA, interactive visualizations, ML predictions, and downloadable reports in one click.

## 🔴 Live Demo
👉 [Click here to try the live app]([STREAMLIT_LINK_YAHAN_PASTE_KARO])

## ✨ Features
- 📂 **Multi-format Input** — CSV, Excel, PDF (digital tables), Image (OCR)
- 📋 **Auto EDA** — KPIs, missing values, duplicates, outliers (IQR method)
- 📈 **Interactive Charts** — Bar, Line, Scatter, Histogram, Pie, Heatmap
- 🤖 **Auto ML** — Select any column as target, model trains automatically (Classification/Regression)
- 📥 **Download Reports** — Excel (6 sheets) + PDF report

## 🛠️ Tech Stack
- **Python** — Pandas, NumPy, Matplotlib, Seaborn, Plotly
- **ML** — Scikit-learn (Random Forest — Classification & Regression)
- **OCR** — Pytesseract, PDFPlumber
- **Reports** — XlsxWriter, ReportLab
- **Deployment** — Streamlit

## 🚀 Run Locally
```bash
git clone https://github.com/krishanchauhan29/smart-analytics-engine.git
cd smart-analytics-engine
pip install -r requirements.txt
streamlit run dashboard/app.py
```
